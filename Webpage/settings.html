Modes

Holiday mode

This is a mode which turns off the power in your home for when you are away for a long period of time


Day mode

This is a mode for everyday living during the day. It limits light use but does not restrict power usage.


Night mode

This is a mode for during the night. It does not limit light usage, but does limit power usage.


Away from home

This is a mode for when you are out of the house - for example during work.
